# 4. 适用场景

Fidus 的架构设计支持多种鲜活的应用场景，以下为典型示例：

## 1. 购物消费
*   **场景**: 买件衣服。
*   **Fidus 作用**: **尺码数据补全**。Agent 自动提供精准的个人尺码数据，避免反复确认或买错。

## 2. 商务办公
*   **场景**: 商务会谈。
*   **Fidus 作用**: **转写 + 自动建库**。自动记录会议内容，转写为文本，提取关键信息存入知识库。

## 3. 专业服务
*   **场景**: 尽职调查 (Due Diligence)。
*   **Fidus 作用**: **尽调提醒，完全填空**。根据框架自动填充已知信息，标记缺失信息，辅助完成报告。

## 4. 社交媒体
*   **场景**: 社媒运营。
*   **Fidus 作用**: **社媒轨迹，KOL -> 粉丝**。分析互动数据，优化内容策略，增强影响力。

## 5. 人脉资源
*   **场景**: 资源对接。
*   **Fidus 作用**: **人脉交换，价值搜索**。基于标签化的人脉网络，快速搜索匹配价值资源。
